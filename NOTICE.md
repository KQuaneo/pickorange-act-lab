# Attribution and scope

This repository contains a research and evaluation layer developed for the
PickOrange task. It is designed to be used with
[LeIsaac](https://github.com/LightwheelAI/leisaac),
[Isaac Lab](https://developer.nvidia.com/isaac/lab), and
[LeRobot](https://github.com/huggingface/lerobot).

LeIsaac is Copyright Lightwheel and the LeIsaac Project Developers and is
licensed under Apache-2.0. This repository retains the same license for the
included derivative experiment scripts. Isaac Lab, LeRobot, and ACT remain
subject to their respective upstream licenses and citation requirements.

No model checkpoints, private datasets, simulator assets, credentials, or
machine-specific runtime directories are included. Paths in the published raw
JSON evidence were converted from machine-specific absolute paths to paths
relative to a LeIsaac project root.
